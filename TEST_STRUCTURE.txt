Test Execution Commands:
========================

Run all tests:
  python run_tests.py
  pytest tests/

Run specific file:
  python run_tests.py -f test_working_memory.py
  pytest tests/test_working_memory.py

Run by marker:
  python run_tests.py -m working
  pytest tests/ -m working

Generate coverage:
  python run_tests.py --coverage
  pytest tests/ --cov=hierarchical_memory --cov-report=html

List tests:
  python run_tests.py --list
  pytest tests/ --collect-only

Quick test (skip slow):
  pytest tests/ -m "not slow"
